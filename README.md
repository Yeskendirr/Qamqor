# QamQor — Благотворительный фонд

## Запуск

### Backend
```bash
cd backend
npm install
npm run dev
# http://localhost:3001
```

### Frontend
```bash
cd frontend
npm install
npm run dev
# http://localhost:3000
```

## Данные для входа в админ-панель
- Логин: `admin`
- Пароль: `admin123`

## Стек
- **Frontend:** React 18, TypeScript, Vite, Tailwind CSS
- **Backend:** Node.js, Express, SQLite (better-sqlite3), JWT
